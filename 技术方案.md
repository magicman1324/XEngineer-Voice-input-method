## 1. 项目概述

### 1.1 项目背景

七牛云 × XEngineer 暑期实训营（第一批次议题实战），题目一：语音输入法。

### 1.2 项目目标

开发一款语音输入法产品，帮助用户提高跨语言场景下的文本输入效率，较好地平衡准确度、易用性、响应速度、成本等关键因素。

### 1.3 核心策略

- **72 小时极限交付**，秉持"小而美"原则
- **零配置前端**：原生 HTML5/CSS3/JavaScript + Tailwind CDN，无构建步骤
- **核心 API 驱动**：火山引擎 ASR + 豆包大模型翻译 + 火山引擎 TTS 实现业务闭环
- **PR 粒度交付**：每个 PR 只做一件事，保持清晰 commit 记录

## 2. 产品定位与场景

### 2.1 产品定位

一款主打「跨语言沟通同传字幕」场景的移动端响应式 Web App (H5)。通过提供"边说边译"的极速体验，大幅提高跨语言环境下的文本输入效率。

### 2.2 目标用户

| 用户类型 | 核心痛点 | 使用频率 |
|----------|---------|---------|
| 跨国会议参与者 | 需要实时理解外方发言 | 高频 |
| 外资企业销售 | 与外国客户沟通时需快速翻译 | 高频 |
| 涉外导游 | 接待外国游客时的即时沟通 | 中频 |
| 留学生/海外工作者 | 日常生活场景的语言辅助 | 中高频 |
| 外语学习者 | 辅助听力理解和口语练习 | 中频 |

### 2.3 核心场景

**面对面跨国交流或在线会议：**
1. 用户打开 H5 页面
2. 手机持续收音并实时展示源语言（如中文）及目标语言（如英文）的同传双语字幕
3. 一键将翻译好的文本复制发送到聊天软件

### 2.4 设备画像

- **目标设备**：现代移动端浏览器
  - iOS Safari 14+
  - Android Chrome 90+
  - 微信内置浏览器（兼容模式）
- **硬件要求**：支持麦克风权限的智能手机
- **网络要求**：4G/5G/WiFi 稳定连接（WebSocket 长连接）

## 4. 技术选型详解

### 4.1 选型总览

| 模块 | 技术选型 | 版本/方案 | 选择理由 |
|------|---------|----------|---------|
| HTML 模板 | HTML5 | W3C 标准 | 零依赖、浏览器原生支持 |
| CSS 框架 | Tailwind CSS | v3 CDN | 无需构建工具，即插即用 |
| JavaScript | Vanilla JS | ES2020+ | 无框架学习成本，极致轻量 |
| 音频捕获 | MediaRecorder API + Web Audio API | W3C 标准 | 浏览器原生，无需第三方 SDK |
| VAD 检测 | 自研轻量 VAD | 音量阈值 + 计时器 | 避免大型 WASM 依赖，< 50 行代码 |
| 语音识别 | 火山引擎流式 ASR | WebSocket 协议 | 低延迟、高并发、竞品中性价比最优 |
| 机器翻译 | 豆包大模型 | ChatCompletions API | 上下文感知翻译，质量优于传统 MT |
| 语音合成 | 火山引擎 TTS | REST API | 高拟真度，与 ASR 同一平台统一管理 |

### 4.2 选型对比分析

#### 前端框架选择

| 维度 | 原生三剑客 + CDN | React/Vue | Next/Nuxt |
|------|-----------------|-----------|-----------|
| 环境搭建 | 0 分钟 | 15-30 分钟 | 30-60 分钟 |
| 首屏加载 | ~50ms | ~200ms | ~500ms |
| JS 体积 | ~0KB（框架代码） | ~40KB (gzip) | ~80KB (gzip) |
| 开发调试 | DevTools 直调 | 需 DevTools + React DevTools | 需更多调试工具 |
| 学习成本 | 已有基础即可 | 需框架知识 | 需全栈知识 |
| 组件复用 | 函数 + 字符串 | 组件化 | 组件化 + SSR |
| **72h 适配** | ★★★★★ | ★★★★ | ★★★ |

**结论**：72h 极限交付 + 界面复杂度不高 → 原生三剑客最优。

#### VAD 方案选择

| 方案 | 准确度 | 加载体积 | 实现复杂度 | 选择 |
|------|-------|---------|-----------|------|
| 自研音量阈值 VAD | ★★★ | ~0.5KB | 极低 | **✓** |
| Silero VAD (WASM) | ★★★★★ | ~2MB | 高 | ✗ |
| WebRTC VAD (deprecated) | ★★★ | 已废弃 | 中 | ✗ |

**结论**：自研音量 VAD 在移动端场景下足够使用，体积趋近于零。

### 6.3 VAD 调度逻辑

```
AudioService.ondata = (pcmChunk, volume) => {
    // 1. 发送到 ASR (持续进行)
    asrService.sendAudio(pcmChunk)

    // 2. 更新 UI 音量指示
    ui.updateVolumeIndicator(volume)

    // 3. VAD 状态判断
    const isSpeech = volume > vad.threshold

    if (isSpeech) {
        vad.resetSilenceTimer()
        vad.isSpeaking = true
    } else {
        vad.silenceTime += chunkDuration
        if (vad.silenceTime > vad.timeout && vad.isSpeaking) {
            // 断句触发
            vad.isSpeaking = false
            asrService.endSegment()
            // 等待 ASR 返回最终结果后触发翻译
        }
    }
}
```

## 7. VAD 静音检测策略

### 7.1 算法设计

采用基于音量阈值的轻量 VAD，无需外部依赖。

```
输入：音频帧 (PCM Int16 样本数组)

步骤：
1. 计算 RMS (Root Mean Square) 音量
   rms = sqrt(sum(samples[i]^2) / len(samples))

2. 转换为分贝 (dB)
   dB = 20 * log10(rms / MAX_INT16)
   其中 MAX_INT16 = 32768

3. 语音判定
   isSpeech = dB > threshold

4. 状态转换
   沉默 → 语音: isSpeech == true  且 持续 > minSpeechDuration (150ms)
   语音 → 沉默: isSpeech == false 且 持续 > silenceTimeout (800ms)
```

### 7.2 参数配置

| 参数 | 默认值 | 说明 | 调优方向 |
|------|--------|------|---------|
| `silenceThreshold` | -45 dB | 静音判定阈值 | 安静环境调低，嘈杂环境调高 |
| `silenceTimeout` | 800 ms | 断句静音超时 | 语速快调低，语速慢调高 |
| `minSpeechDuration` | 150 ms | 最小语音段长度 | 过滤短促噪音 |
| `minSegmentDuration` | 300 ms | 最小有效句长度 | 过滤呼吸声、点击声 |
| `frameSize` | 2048 样本 | FFT 帧大小 | 影响音量计算精度 |
| `dynamicThreshold` | true | 是否启用动态阈值 | 环境自适应 |

### 7.3 动态阈值校准

```
首次启动时：
1. 收集环境噪音 500ms
2. 计算噪音平均 dB：noiseFloor
3. 设置阈值：threshold = noiseFloor + 15dB (信噪比要求)

运行过程中：
1. 每 5 秒重新评估一次噪音底噪
2. 在非语音段更新 noiseFloor
3. 滑动窗口平均：newNoiseFloor = 0.7 * oldNoiseFloor + 0.3 * currentNoise
4. 避免阈值剧烈波动：变化幅度 ≤ 3dB/次
```

### 7.4 抗噪处理

| 问题 | 解决方案 |
|------|---------|
| 环境底噪 | 动态阈值校准 + 高通滤波 (HPF, cut-off 80Hz) |
| 突发噪音（关门声） | 能量突变检测：若音量突变 > 30dB 且持续时间 < 50ms，标记为噪音 |
| 呼吸声 | 最小语音段长度过滤 (150ms) |
| 麦克风爆音 | 削波检测：若连续样本 > 32000 的比例 > 5%，标记为爆音段 |
| 多人同时说话 | 不做复杂处理（同传场景以主讲人为主） |

#### ASR 服务实现关键点

```javascript
// asr.js 核心设计

class ASRService {
    constructor(config) {
        this.wsUrl = config.asrWsUrl
        this.appid = config.asrAppid
        this.token = config.asrToken
        this.ws = null
        this.reconnectAttempts = 0
        this.maxReconnectAttempts = 3
        this.reconnectDelay = 1000
        this.heartbeatInterval = null
        this.onIntermediateResult = null  // 中间结果回调 (灰字)
        this.onFinalResult = null         // 最终结果回调 (黑字)
        this.onError = null
    }

    async connect() {
        // 1. 建立 WebSocket 连接
        this.ws = new WebSocket(this.wsUrl)
        this.ws.binaryType = 'arraybuffer'

        // 2. 连接成功后发送认证请求
        this.ws.onopen = () => {
            this.sendAuthRequest()
            this.startHeartbeat()
            this.reconnectAttempts = 0
        }

        // 3. 处理服务端消息
        this.ws.onmessage = (event) => {
            this.handleMessage(event.data)
        }

        // 4. 断线重连
        this.ws.onclose = () => {
            this.stopHeartbeat()
            this.attemptReconnect()
        }

        this.ws.onerror = (err) => {
            this.onError && this.onError(err)
        }
    }

    sendAudio(pcmChunk) {
        if (this.ws && this.ws.readyState === WebSocket.OPEN) {
            this.ws.send(pcmChunk)
        }
    }

    endSegment() {
        // 发送结束标志，等待最终结果
        const endMsg = JSON.stringify({
            request: { is_final: true }
        })
        this.ws && this.ws.send(endMsg)
    }

    handleMessage(data) {
        if (typeof data === 'string') {
            const msg = JSON.parse(data)
            if (msg.code !== 0) {
                this.onError && this.onError(msg)
                return
            }
            if (msg.type === 0) {
                // 中间结果
                this.onIntermediateResult && this.onIntermediateResult(msg.result[0].text)
            } else if (msg.type === 1 || msg.is_final) {
                // 最终结果
                this.onFinalResult && this.onFinalResult(msg.result[0].text)
            }
        }
    }

    startHeartbeat() {
        this.heartbeatInterval = setInterval(() => {
            // 发送心跳保持连接
            if (this.ws.readyState === WebSocket.OPEN) {
                this.ws.send(JSON.stringify({ type: 'heartbeat' }))
            }
        }, 10000) // 每 10s 一次
    }

    stopHeartbeat() {
        clearInterval(this.heartbeatInterval)
    }

    attemptReconnect() {
        if (this.reconnectAttempts < this.maxReconnectAttempts) {
            this.reconnectAttempts++
            setTimeout(() => this.connect(), this.reconnectDelay * this.reconnectAttempts)
        } else {
            this.onError && this.onError(new Error('ASR 连接失败，已超出最大重试次数'))
        }
    }

    disconnect() {
        this.stopHeartbeat()
        this.ws && this.ws.close()
    }
}
```

```javascript
// translator.js 核心设计

class Translator {
    constructor(config) {
        this.apiUrl = config.translateApiUrl
        this.apiKey = config.translateApiKey
        this.model = config.translateModel || 'doubao-1.5-pro-256k'
        this.cache = new Map()  // 相同文本去重缓存
        this.requestQueue = []
        this.isProcessing = false
        this.maxRetries = 2
        this.timeout = 5000
    }

    async translate(text, sourceLang, targetLang) {
        // 1. 缓存检查
        const cacheKey = `${sourceLang}:${targetLang}:${text}`
        if (this.cache.has(cacheKey)) {
            return this.cache.get(cacheKey)
        }

        // 2. 空文本检查
        if (!text || text.trim().length === 0) {
            return ''
        }

        // 3. 发送翻译请求
        try {
            const result = await this.callTranslateAPI(text, sourceLang, targetLang)
            // 4. 写入缓存
            this.cache.set(cacheKey, result)
            // 限制缓存大小防止内存泄露
            if (this.cache.size > 100) {
                const firstKey = this.cache.keys().next().value
                this.cache.delete(firstKey)
            }
            return result
        } catch (error) {
            console.error('翻译失败:', error)
            throw error
        }
    }

    async callTranslateAPI(text, sourceLang, targetLang) {
        const controller = new AbortController()
        const timeoutId = setTimeout(() => controller.abort(), this.timeout)

        try {
            const response = await fetch(this.apiUrl, {
                method: 'POST',
                headers: {
                    'Content-Type': 'application/json',
                    'Authorization': `Bearer ${this.apiKey}`
                },
                body: JSON.stringify({
                    model: this.model,
                    messages: [
                        {
                            role: 'system',
                            content: this.buildSystemPrompt(sourceLang, targetLang)
                        },
                        {
                            role: 'user',
                            content: text
                        }
                    ],
                    temperature: 0.3,
                    max_tokens: 1024
                }),
                signal: controller.signal
            })

            if (!response.ok) {
                throw new Error(`翻译 API 错误: ${response.status} ${response.statusText}`)
            }

            const data = await response.json()
            return data.choices[0].message.content.trim()
        } finally {
            clearTimeout(timeoutId)
        }
    }

    buildSystemPrompt(sourceLang, targetLang) {
        return `你是一个专业翻译引擎。请将用户输入的内容从${sourceLang}翻译为${targetLang}。要求：1) 保持原意和语气；2) 专业术语翻译准确；3) 输出仅包含译文，不要有任何额外说明；4) 如果输入包含多个句子，保持句子结构对应。`
    }
}
```

```javascript
/**
 * 音频管理模块
 * 职责：麦克风权限管理、音频流捕获、音量分析
 */

class AudioService {
    constructor(config) {
        this.config = config
        this.stream = null
        this.mediaRecorder = null
        this.audioContext = null
        this.analyser = null
        this.source = null
        this.audioChunks = []
        this.isRecording = false
        this.onAudioData = null      // 回调：PCM 数据 + 音量
        this.onVolumeChange = null   // 回调：音量变化
        this.onError = null
        this.animationId = null
    }

    // 请求麦克风权限
    async requestMicrophone() {
        try {
            this.stream = await navigator.mediaDevices.getUserMedia({
                audio: {
                    sampleRate: this.config.audio.sampleRate,
                    channelCount: this.config.audio.channelCount,
                    echoCancellation: true,
                    noiseSuppression: true,
                    autoGainControl: true,
                }
            })
            return true
        } catch (err) {
            if (err.name === 'NotAllowedError') {
                throw new Error('麦克风权限被拒绝，请在浏览器设置中允许麦克风访问')
            } else if (err.name === 'NotFoundError') {
                throw new Error('未检测到麦克风设备')
            } else {
                throw new Error(`麦克风访问失败: ${err.message}`)
            }
        }
    }

    // 初始化 Web Audio 管道
    initAudioPipeline() {
        this.audioContext = new (window.AudioContext || window.webkitAudioContext)()
        this.source = this.audioContext.createMediaStreamSource(this.stream)
        this.analyser = this.audioContext.createAnalyser()
        this.analyser.fftSize = this.config.vad.frameSize
        this.analyser.smoothingTimeConstant = 0.8

        this.source.connect(this.analyser)

        // 初始化 MediaRecorder
        const mimeType = this.getSupportedMimeType()
        this.mediaRecorder = new MediaRecorder(this.stream, {
            mimeType: mimeType,
            audioBitsPerSecond: this.config.audio.sampleRate * 16,
        })

        this.mediaRecorder.ondataavailable = (event) => {
            if (event.data.size > 0) {
                this.audioChunks.push(event.data)
                const volume = this.getCurrentVolume()
                this.onAudioData && this.onAudioData(event.data, volume)
            }
        }
    }

    // 获取支持的音频 MIME 类型
    getSupportedMimeType() {
        const types = [
            'audio/webm;codecs=opus',
            'audio/webm',
            'audio/ogg;codecs=opus',
            'audio/mp4',
        ]
        for (const type of types) {
            if (MediaRecorder.isTypeSupported(type)) {
                return type
            }
        }
        return '' // 浏览器默认
    }

    // 开始录音
    startRecording() {
        if (this.mediaRecorder && this.mediaRecorder.state === 'inactive') {
            this.audioChunks = []
            this.mediaRecorder.start(200) // 每 200ms 触发一次 ondataavailable
            this.isRecording = true
            this.startVolumeMonitoring()
        }
    }

    // 停止录音
    stopRecording() {
        if (this.mediaRecorder && this.mediaRecorder.state === 'recording') {
            this.mediaRecorder.stop()
            this.isRecording = false
            this.stopVolumeMonitoring()
        }
    }

    // 获取当前音量 (RMS)
    getCurrentVolume() {
        if (!this.analyser) return -100
        const dataArray = new Uint8Array(this.analyser.frequencyBinCount)
        this.analyser.getByteTimeDomainData(dataArray)

        let sum = 0
        for (let i = 0; i < dataArray.length; i++) {
            const value = (dataArray[i] - 128) / 128
            sum += value * value
        }
        const rms = Math.sqrt(sum / dataArray.length)
        const db = rms > 0 ? 20 * Math.log10(rms) : -100

        return Math.max(-100, Math.min(0, db))
    }

    // 音量监控循环
    startVolumeMonitoring() {
        const monitor = () => {
            if (!this.isRecording) return
            const volume = this.getCurrentVolume()
            this.onVolumeChange && this.onVolumeChange(volume)
            this.animationId = requestAnimationFrame(monitor)
        }
        monitor()
    }

    stopVolumeMonitoring() {
        if (this.animationId) {
            cancelAnimationFrame(this.animationId)
            this.animationId = null
        }
    }

    // 释放资源
    release() {
        this.stopRecording()
        if (this.stream) {
            this.stream.getTracks().forEach(track => track.stop())
            this.stream = null
        }
        if (this.audioContext) {
            this.audioContext.close()
            this.audioContext = null
        }
    }
}
```

### 10.3 vad.js — 语音活动检测

```javascript
/**
 * VAD 模块
 * 职责：基于音量的轻量语音活动检测
 */

class VAD {
    constructor(config) {
        this.config = config.vad
        this.isSpeaking = false
        this.silenceTime = 0
        this.segmentStartTime = 0
        this.speechDuration = 0
        this.noiseFloor = -50  // 初始噪音底噪
        this.noiseSamples = []
        this.isCalibrating = true
        this.calibrationCount = 0
        this.dynamicThreshold = this.config.dynamicThreshold
        this.onSpeechStart = null
        this.onSpeechEnd = null
        this.onSegmentReady = null
    }

    // 处理每个音频帧的音量
    process(volume, timestamp) {
        // 动态阈值校准
        this.calibrateNoiseFloor(volume)

        const threshold = this.getThreshold()
        const isSpeech = volume > threshold

        if (isSpeech) {
            if (!this.isSpeaking) {
                // 检查语音持续时间是否超过最短语音段
                this.speechDuration = 0
            }
            this.isSpeaking = true
            this.silenceTime = 0
            this.speechDuration += this.getFrameDuration()

            // 从沉默转为语音
            if (!this.isSpeaking) {
                this.isSpeaking = true
                this.segmentStartTime = timestamp
                this.onSpeechStart && this.onSpeechStart()
            }
        } else {
            if (this.isSpeaking) {
                this.silenceTime += this.getFrameDuration()

                // 静音超过阈值 → 断句
                if (this.silenceTime >= this.config.silenceTimeout / 1000) {
                    // 检查句子是否足够长
                    if (this.speechDuration >= this.config.minSegmentDuration / 1000) {
                        this.onSegmentReady && this.onSegmentReady()
                    }
                    this.isSpeaking = false
                    this.silenceTime = 0
                    this.speechDuration = 0
                    this.onSpeechEnd && this.onSpeechEnd()
                }
            }
        }
    }

    // 校准噪音底噪
    calibrateNoiseFloor(volume) {
        if (!this.dynamicThreshold) return

        if (!this.isSpeaking) {
            this.noiseSamples.push(volume)
            if (this.noiseSamples.length > 100) {
                this.noiseSamples.shift()
            }

            if (this.isCalibrating && this.noiseSamples.length >= 10) {
                this.isCalibrating = false
            }

            // 滑动窗口计算噪音底噪
            if (this.noiseSamples.length >= 10) {
                const sorted = [...this.noiseSamples].sort((a, b) => a - b)
                const median = sorted[Math.floor(sorted.length / 2)]
                this.noiseFloor = 0.7 * this.noiseFloor + 0.3 * median
            }
        }
    }

    // 获取当前动态阈值
    getThreshold() {
        if (this.dynamicThreshold && !this.isCalibrating) {
            return Math.max(
                this.noiseFloor + this.config.thresholdMargin,
                this.config.silenceThreshold
            )
        }
        return this.config.silenceThreshold
    }

    getFrameDuration() {
        return this.config.frameSize / 16000 // 近似帧时长（秒）
    }

    reset() {
        this.isSpeaking = false
        this.silenceTime = 0
        this.speechDuration = 0
        this.segmentStartTime = 0
        this.noiseSamples = []
        this.isCalibrating = true
    }
}
```

```javascript
/**

class App {
    constructor(config) {
        this.config = config
        this.state = STATES.IDLE
        this.stateCallbacks = new Map()

        // 子模块
        this.audio = null
        this.vad = null
        this.asr = null
        this.translator = null
        this.tts = null
        this.ui = null

        // 数据
        this.currentSourceText = ''
        this.currentTargetText = ''
        this.segments = []       // 已完成的翻译段落
        this.interimText = ''    // 当前 ASR 中间结果
    }

    async initialize() {
        // 1. 初始化各模块
        this.audio = new AudioService(this.config)
        this.vad = new VAD(this.config)
        this.asr = new ASRService(this.config)
        this.translator = new Translator(this.config)
        this.tts = new TTSService(this.config)
        this.ui = new UIManager(this.config)

        // 2. 注册 VAD 回调
        this.vad.onSpeechStart = () => {
            this.ui.setStatus('speaking')
        }

        this.vad.onSegmentReady = () => {
            this.setState(STATES.PROCESSING)
            this.asr.endSegment()
        }

        // 3. 注册 ASR 回调
        this.asr.onIntermediateResult = (text) => {
            this.interimText = text
            this.ui.updateSourceText(this.segments, this.interimText, false)
        }

        this.asr.onFinalResult = async (text) => {
            this.currentSourceText = text
            this.interimText = ''
            // 显示最终定稿文本
            this.ui.updateSourceText(this.segments, text, true)

            // 触发翻译
            await this.performTranslation(text)
        }

        this.asr.onError = (error) => {
            this.handleError(error)
        }

        // 4. 注册音频回调
        this.audio.onAudioData = (chunk, volume) => {
            // 发送音频到 ASR
            this.asr.sendAudio(chunk)
            // VAD 处理
            this.vad.process(volume, Date.now())
        }

        this.audio.onVolumeChange = (volume) => {
            this.ui.updateVolumeIndicator(volume)
        }

        // 5. 初始化 UI
        this.ui.render()
        this.ui.bindEvents(this)
    }

    // 切换状态
    setState(newState) {
        const oldState = this.state
        this.state = newState
        this.ui.updateUIForState(newState, oldState)

        // 触发状态变更回调
        if (this.stateCallbacks.has(newState)) {
            this.stateCallbacks.get(newState).forEach(cb => cb())
        }
    }

    // 开始录音
    async startListening() {
        try {
            this.setState(STATES.REQUESTING_MIC)
            await this.audio.requestMicrophone()
            this.audio.initAudioPipeline()

            // 连接 ASR
            await this.asr.connect()

            // 开始录音
            this.audio.startRecording()
            this.setState(STATES.LISTENING)
        } catch (err) {
            if (err.message.includes('权限被拒绝')) {
                this.setState(STATES.MIC_DENIED)
            } else {
                this.handleError(err)
            }
        }
    }

    // 停止录音
    stopListening() {
        this.audio.stopRecording()
        this.asr.endSegment()    // 等待最后一段结果
        setTimeout(() => {
            this.asr.disconnect()
            this.audio.release()
            this.vad.reset()
            this.setState(STATES.IDLE)
        }, 500)
    }

    // 执行翻译
    async performTranslation(text) {
        try {
            this.setState(STATES.TRANSLATING)
            const translated = await this.translator.translate(
                text,
                this.config.languages.default.source,
                this.config.languages.default.target
            )

            this.currentTargetText = translated
            this.segments.push({
                source: text,
                target: translated,
                timestamp: Date.now()
            })

            this.ui.updateTargetText(this.segments)
            this.setState(STATES.LISTENING)
        } catch (err) {
            this.handleError(err)
        }
    }

    // 错误处理
    handleError(error) {
        console.error('[App Error]', error)
        this.setState(STATES.ERROR)
        this.ui.showToast(error.message || '发生未知错误', 'error')

        // 3 秒后自动恢复
        setTimeout(() => {
            if (this.state === STATES.ERROR) {
                this.setState(STATES.IDLE)
            }
        }, 3000)
    }

    // TTS 朗读翻译结果
    async playTTS(text, lang) {
        try {
            this.setState(STATES.PLAYING_TTS)
            await this.tts.speak(text, lang)
            // 恢复之前的状态
            if (this.audio.isRecording) {
                this.setState(STATES.LISTENING)
            } else {
                this.setState(STATES.IDLE)
            }
        } catch (err) {
            this.handleError(err)
        }
    }

    // 复制文本到剪贴板
    async copyText(text) {
        try {
            await navigator.clipboard.writeText(text)
            this.ui.showToast('已复制到剪贴板', 'success')
        } catch {
            // 降级方案：使用 textarea
            const textarea = document.createElement('textarea')
            textarea.value = text
            document.body.appendChild(textarea)
            textarea.select()
            document.execCommand('copy')
            document.body.removeChild(textarea)
            this.ui.showToast('已复制到剪贴板', 'success')
        }
    }

    // 切换语言
    setLanguage(source, target) {
        this.config.languages.default.source = source
        this.config.languages.default.target = target
        this.ui.updateLanguageIndicator(source, target)
    }

    // 清空会话
    clearSession() {
        this.segments = []
        this.currentSourceText = ''
        this.currentTargetText = ''
        this.interimText = ''
        this.ui.clearDisplay()
    }
}
```

```javascript
            toast.classList.add('opacity-100')
        })
        setTimeout(() => {
            toast.classList.remove('opacity-100')
            toast.classList.add('opacity-0')
        }, 2500)
    }

    // 事件绑定
    bindEvents(app) {
        this.elements.micBtn.addEventListener('click', () => {
            if (app.state === STATES.IDLE || app.state === STATES.ERROR || app.state === STATES.MIC_DENIED) {
                app.startListening()
            } else {
                app.stopListening()
            }
        })

        this.elements.copyBtn.addEventListener('click', () => {
            const text = app.segments.map(s => `[${s.source}] ${s.target}`).join('\n')
            if (text) app.copyText(text)
            else this.showToast('没有可复制的内容', 'info')
        })

        this.elements.ttsBtn.addEventListener('click', () => {
            const text = app.segments.map(s => s.target).join(' ')
            if (text) app.playTTS(text, app.config.languages.default.target)
            else this.showToast('没有可朗读的内容', 'info')
        })

        this.elements.clearBtn.addEventListener('click', () => {
            app.clearSession()
        })

        this.elements.langSwitchBtn.addEventListener('click', () => {
            this.showLanguageModal(app)
        })
    }

    // HTML 转义
    escapeHtml(text) {
        const div = document.createElement('div')
        div.textContent = text
        return div.innerHTML
    }
}
```

```javascript
// app.js 中的全局错误处理
function setupGlobalErrorHandling(app) {
    // 未捕获的 Promise 异常
    window.addEventListener('unhandledrejection', (event) => {
        console.error('[Unhandled Rejection]', event.reason)
        app.handleError(event.reason)
    })

    // 全局异常
    window.onerror = (message, source, lineno, colno, error) => {
        console.error('[Global Error]', message, error)
        app.handleError(error || new Error(message))
    }

    // 网络状态监听
    window.addEventListener('offline', () => {
        app.handleError(new Error('网络连接已断开'))
    })

    window.addEventListener('online', () => {
        app.ui.showToast('网络已恢复', 'success')
        if (app.state === STATES.ERROR) {
            app.setState(STATES.IDLE)
        }
    })

    // 页面可见性变化 (切到后台时暂停录音)
    document.addEventListener('visibilitychange', () => {
        if (document.hidden && app.audio && app.audio.isRecording) {
            app.audio.stopRecording()
        }
    })
}
```

## 15. 安全方案

### 15.1 前端安全

| 风险 | 防护措施 |
|------|---------|
| API Key 泄露 | API Key 仅存储在服务端或受限环境，前端使用代理转发 |
| XSS | 所有用户输入和 API 返回文本经 `textContent` 或 `escapeHtml` 处理 |
| 中间人攻击 | 全站 HTTPS / WSS，不降级到 HTTP |
| 敏感音频数据 | 音频流仅发送到火山引擎，不在本地持久化存储 |

### 15.2 API 安全

建议通过一个轻量代理服务（云函数 / Vercel Edge）转发 API 请求，避免将 API Key 暴露在前端：

```
    │  请求 (无 Key)          │                          │

    │                        │────────────────────────>│
```
